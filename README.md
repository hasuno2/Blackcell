# Blackcell

Blackcell is an umbrella workspace for small command-line utilities.

## Projects
- **watchdog/** - interactive bash session logger. See `watchdog/README.md` for full instructions.

## Quick start
```
git clone <repo-url>
cd Blackcell/watchdog
pip install -e .
```

More tools will land under this repository over time. Each subproject is self-contained with its own packaging metadata.
